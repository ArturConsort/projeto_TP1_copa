# Projeto TP1 - Sistema de Gestão da Copa do Mundo 2026 ⚽

## 📖 Sobre o Projeto
O sistema é uma aplicação desenvolvida para gerenciar e simular os dados da Copa do Mundo de 2026. Ele oferece uma interface gráfica intuitiva para o controle de seleções, partidas, estádios e ingressos, garantindo uma experiência de gestão completa do torneio.

Este projeto foi submetido como trabalho final da disciplina Técnicas de Programação 1, ministrada pela Dra. Prof. Roberta Barbosa Oliveira no primeiro semestre de 2026.

## 💻 Tecnologias Utilizadas
O programa foi desenvolvido em Java por meio da IDE IntelliJ e do gerenciador de dependências Maven. A interface gráfica para o usuário foi implementada usando JavaFX.
